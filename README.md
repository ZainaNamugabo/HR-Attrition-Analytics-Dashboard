# HR-Attrition-Analytics-Dashboard
I designed and developed an HR Attrition Analytics Dashboard to help organizations better understand employee turnover, workforce composition, and retention patterns through data.

This dashboard transforms complex HR data into clear, actionable insights that support smarter decision-making for HR teams and leadership.

At a glance, it provides a high-level overview of key workforce metrics, including total employees (3,000), active employees (1,467), ex-employees (1,533), attrition rate (51.10%), and average age (54.04)—giving stakeholders an immediate understanding of organizational health.

# Key Features & Insights
Employee Demographics
Visual breakdown of male vs female employee percentages for diversity insights

Attrition by Age Group
Identifies which age segments (young, middle-aged, senior) experience the highest turnover

Attrition Trends Over Time
By start year: reveals hiring vs attrition patterns
By exit year: highlights periods of increased employee exits

Department-Level Analysis
Pinpoints which departments (e.g., Production, IT, Sales) have the highest attrition rates

Performance-Based Attrition
Understand whether top performers or underperformers are more likely to leave

Interactive Filters
Users can drill down by state, race, and job title for deeper analysis
